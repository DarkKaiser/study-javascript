jQuery란?

jQuery는 가볍고 빠르며, 간결한 오픈소스 스크립트 라이브러리입니다. 이를 이용하면 Rich 웹 UI를 개발하는 데 도움이 되는 다양한 기능들 즉, HTML 문서 트래버스, 이벤트 처리, 애니메이션, Ajax 상호작용 등을 지원하여 빠르고 견고하게 리치 웹 애플리케이션 개발을 할 수 있도록 지원합니다.

이러한 jQuery의 기능적인 특징은 다음과 같습니다

  • 막강한 CSS 셀렉터
  • 크로스 브라우저 지원
  • 메서드 체인
  • Ajax 지원
  • 풍부한 플러그 인 지원
